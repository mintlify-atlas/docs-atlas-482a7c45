## Overview

The User Management Microservice is a demonstration project showcasing best practices in Spring Boot API development. It implements user registration and authentication using JWT tokens, built on hexagonal architecture principles for clean separation of concerns.

## Technology Stack

This microservice is built with:

- **Spring Boot** - Application framework
- **Spring Security** - Authentication and authorization
- **Spring Data JPA** - Data persistence
- **H2 Database** - In-memory database for development
- **JWT (JJWT)** - Token-based authentication
- **Gradle** - Build automation
- **Lombok** - Boilerplate reduction
- **MapStruct** - Object mapping
